# IPv4

IPv4地址大小为4个字节，用点分十进制表示法可以表达范围`0.0.0.0`到`255.255.255.255`的地址。

## 地址分类

最初，IPv4地址空间分为5大类。

![ip-5-class](ip-5-class.png)

|  类别   |           范围            |                       说明                        |
| :-----: | :-----------------------: | :-----------------------------------------------: |
| A类地址 |  0.0.0.0-127.255.255.255  |   用于少量的大型网络，第一个字节的最高位固定为0   |
| B类地址 | 128.0.0.0-191.255.255.255 | 用于中等规模的网络，第一个字节的最高2比特固定为10 |
| C类地址 | 192.0.0.0-223.255.255.255 | 用于小规模的网络，第一个字节的最高3比特固定为110  |
| D类地址 | 224.0.0.0-239.255.255.255 |                     组播地址                      |
| E类地址 | 240.0.0.0-255.255.255.255 |                     保留地址                      |

## 无类别域间路由（Classless Inter-Domain Routing CIDR）

最初的IPv4地址是分为A,B,C类的，但是在实际使用中有很大的问题。分配一个A类网络可容纳1677万台主机，显得太多，C类网络只容纳254台主机，又显得太少。很多时候C类网络就够用了，但是C类网络多了，由于地域太分散不好聚合路由又导致路由信息大量增加。

无类别域间路由（CIDR）允许对采用可变长子网掩码（VLSM）的网络进行聚合。解决了主机数量不灵活和路由表膨胀的两个关键问题。

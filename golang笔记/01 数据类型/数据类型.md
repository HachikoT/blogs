# 基本类型

|       类型       |        说明         |           占用内存           | 默认初始值 |
| :--------------: | :-----------------: | :--------------------------: | :--------: |
|      `bool`      |       布尔型        |            1字节             |  `false`   |
|      `int`       |     有符号整型      | 32位系统4字节，64位系统8字节 |    `0`     |
|      `uint`      |     无符号整型      | 32位系统4字节，64位系统8字节 |    `0`     |
| `int8/16/32/64`  |   有符号定长整型    |         1/2/4/8字节          |    `0`     |
| `uint8/16/32/64` |   无符号定长整型    |         1/2/4/8字节          |    `0`     |
|    `uintptr`     | 能容纳指针值的整型  | 32位系统4字节，64位系统8字节 |    `0`     |
|      `byte`      |      字节类型       |            1字节             |    `0`     |
|      `rune`      | 存储unicode码点类型 |            4字节             |    `0`     |
|    `float32`     |    单精度浮点型     |            4字节             |   `0.0`    |
|    `float64`     |    双精度浮点型     |            8字节             |   `0.0`    |
|   `complex64`    |    单精度复数型     |            8字节             | `0.0+0.0i` |
|   `complex128`   |    双精度复数型     |            16字节            | `0.0+0.0i` |
|     `string`     |     字符串类型      |              -               |    `""`    |

- **整型字面值常量默认为`int`类型，浮点型字面值常量默认为`float64`类型**
- **`byte`是`uint8`的别名（`type byte = uint8`），表示一个字节**
- **`rune`是`int32`的别名（`type rune = int32`），表示一个unicode码点**
- **`real`和`imag`内置函数可以获取复数的实部和虚部**
- **`string`是不可修改的字节序列，默认表示utf-8编码字符串**

# 派生类型

|            类型            |    说明    |            是否可比较（编译时）            |    默认初始值    |
| :------------------------: | :--------: | :----------------------------------------: | :--------------: |
|            `*T`            |  指针类型  |                   可比较                   |      `nil`       |
|           `[N]T`           |  数组类型  |             T可比较，则可比较              | 元素均为其默认值 |
|           `[]T`            |  切片类型  |              只能跟`nil`比较               |      `nil`       |
|         `map[K]V`          |  字典类型  |              只能跟`nil`比较               |      `nil`       |
|      `type T struct`       | 结构体类型 |            成员均可比较则可比较            | 成员均为其默认值 |
|     `type T interface`     |  接口类型  | 一种接口类型包含另一种接口类型接口则可比较 |      `nil`       |
|          `chan T`          |  通道类型  |                   可比较                   |      `nil`       |
| `func(ARGS...) RESULTS...` |  函数类型  |              只能跟`nil`比较               |      `nil`       |

- **指针类型`*T`不支持算术运算，无法对一个指针加上或减去一个数变为另一个指针**
- **数组类型`[N]T`中的元素数量也是类型的一部分，也就是说`[2]int`和`[3]int`是2种不同的类型**
- **字典类型`map[K]V`中键类型`K`必须是可比较的**

# 类型定义和类型别名

类型定义是通过`type`关键字来创建新的类型。这意味着你可以为新类型添加新的方法，而不会影响到原类型。

```go
type myint int

func (n myint) isPositive() bool {
	return n > 0
}

func main() {
	var n myint = 10
	fmt.Println(reflect.TypeOf(n), n.isPositive()) // 输出 main.myint true
}
```

类型别名则是通过`type`关键字为已存在的类型创建一个新的名字。这种方式并不会创建新的底层类型，而是创建了一个现有类型的“别名”。

```go
type myint = int

func main() {
	var n myint = 10
	fmt.Println(reflect.TypeOf(n)) // 输出 int
}
```
